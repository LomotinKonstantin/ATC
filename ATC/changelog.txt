v1.6	(12.04.2019)
--------------------
Добавлен режим сервера.
Для запуска в режиме сервера необходимо выполнить команду
python ATC.py server <port>
где <порт> - номер порта.
В режиме сервера ATC принимает строки json с полями body, language и rubricator, и возвращает либо строку с ошибкой, либо строку json, где ключи - рубрики, а значения - вероятности

v1.3	(07.10.2017
------------------
Добавлен модуль классификатора lr_50_v2. Этот модуль содержит, обученную на сбалансированной выборке по рубрикатору IPV
Для выбора модуля в качестве активного
* в графическом режиме
  1. Нажать на кнопку "Менеджер модулей" на панели инструментов
  2. Переключиться на вкладку "Классификаторы"
  3. Выбрать модуль и нажать кнопку "Ок"
* вручную
  В файле ATC/config.ini заменить свойство "classifier_import" на имя папки с модулем без кавычек (на данный момент либо classifier_v1, либо lr_50_v2)
